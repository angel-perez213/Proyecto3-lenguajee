# Proyecto Final — Simulador de Máquina de Turing + Regex Lab

Aplicación en Python 3 con Tkinter (GUI simple) para simular máquinas de Turing y probar 10 expresiones regulares mediante coincidencia completa (full match). Compatible con Windows, macOS y Linux.

---

## Requisitos

- Python 3.8 o superior.
- Tkinter (incluido en el Python oficial). En algunas distribuciones Linux puede requerir instalación manual.

---

## Cómo obtener el proyecto (dos opciones)

### Opción A: Sin usar Git (para quien no sabe clonar)
1. Entra a la página del repositorio en GitHub.
2. Haz clic en el botón verde que dice **Code**.
3. Selecciona **Download ZIP**.
4. Cuando termine la descarga, descomprime el archivo ZIP en una carpeta de tu preferencia (por ejemplo, Escritorio).
5. Verifica que dentro de la carpeta esté el archivo `Proyectofinal.py`.

### Opción B: Usando Git (clonado)
